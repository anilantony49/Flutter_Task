# Flutter Login and Registration App

This is a Flutter-based mobile application that provides user authentication functionality using Firebase Authentication, including Login and Register features. Users can create a new account using their email and password and securely log in to the application after registration.

The app focuses on a clean and smooth UI, form validation, and smooth animations to provide a better user experience. It ensures seamless navigation between the login and registration screens. The application is designed following Flutter best practices to ensure good performance and maintainable code.

The app is developed using Provider for state management.

Key Features

User registration with email and password

Secure login authentication

Input validation and error handling

Clean UI with smooth animations

Simple and user-friendly interface
